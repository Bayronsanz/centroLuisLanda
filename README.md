Centro Educativo Luis Landa - Sitio Web

Este proyecto consiste en el desarrollo de un sitio web informativo para el Centro Educativo Luis Landa, una escuela pública rural ubicada en Pie de La Cuesta, Gualaco, Olancho, Honduras.

Objetivo

El sitio web tiene como propósito:

Dar visibilidad a la escuela y su historia.
Mostrar el impacto educativo en la comunidad.
Compartir actividades y eventos escolares.
Facilitar el contacto y apoyo mediante donaciones.
Estructura del sitio

El sitio incluye las siguientes secciones:

Inicio: Presentación general de la escuela
Historia: Historia, misión, visión y preguntas frecuentes
Galería: Fotos y actividades escolares
Donaciones: Información sobre cómo apoyar la escuela
Contacto: Datos de contacto de la institución
Tecnologías utilizadas
HTML
CSS
JavaScript
Cómo ejecutar el proyecto
Clonar el repositorio:
git clone https://github.com/Bayronsanz/centroLuisLanda.git
Abrir la carpeta del proyecto:
cd centroLuisLanda
Abrir el archivo index.html en el navegador.
